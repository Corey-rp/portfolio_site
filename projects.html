MSA Secure Pickup Package

A complete three-app student security system built for the Morehouse College STEM Achievers Summer Program. Includes a Parent Registration Portal (QR code + alternate pickup codes), an Admin Dashboard (staff management, session controls, brute-force protection), and a Mentor Scanner App (live QR scanning, Twilio SMS alerts). Hosted across three subdomains on Cloudflare Pages with a Supabase/PostgreSQL backend, automated daily resets via pg_cron, and EmailJS notifications.
